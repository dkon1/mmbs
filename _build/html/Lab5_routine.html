Part 1: Harmonic oscillator

We saw in class that we can write the 2nd order harmonic oscillator ODE

\[m \ddot x = -k x \]

as two 1st order ODEs (setting mass to 1):

\[ dy/dt = -k x \]
\[ dx/dt = y \]
